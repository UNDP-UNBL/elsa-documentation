User manual for the Prioritizing Nature webtool
===============================================

This user's manual aims to provide a clear and concise overview on the use of the Prioritizing Nature web tool, developed within the framework of the `Prioritizing Nature <https://www.learningfornature.org/es/prioritizing-nature-supporting-colombia-ecuador-and-peru-to-use-spatial-data-to-protect-30-of-land-and-seas-by-2030/>`_ project in Ecuador and Peru.
